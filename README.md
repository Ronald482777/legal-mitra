## 🛠 How legal-mitra Works

The app is built with strong tools to help you manage legal data:

- **Next.js** and **NestJS** create the user interface and server logic.
- **PostgreSQL** stores your case data safely.
- **Redis** helps with quick data retrieval and notifications.
- **Blockchain** technology secures your evidence and case activity logs.
- The AI system supports you by answering questions and helping draft documents.

You do not need to worry about these details. The app does all this in the background.
